Aktifkan JavaScript untuk terus melanjutkan ke Website

Browser yang Anda gunakan telah menonaktifkan JavaScript. Untuk melanjutkan ke Website, aktifkan JavaScript.

  1. Klik ⋮, lalu pilih Setelan.
  2. Pilih Privasi dan Keamanan.
  3. Klik Setelan Situs.
  4. Klik JavaScript.
  5. Pilih Situs dapat menggunakan JavaScript.
    (Anda dapat mengaktifkan JavaScript hanya untuk fatonyahmadfauzi.netlify.app)
  1. Klik menu Lainnya ⋮, lalu Setelan.
  2. Pilih Cookie dan izin situs.
  3. Pilih JavaScript.
  4. Pilih Diizinkan (Direkomendasikan).
  1. Ketik about:config di kolom URL, lalu tekan Return.
  2. Klik “Setujui Risiko dan Lanjutkan” untuk melanjutkan.
  3. Temukan preferensi javascript.enabled.
  4. Klik Tombol atau tombol Reset untuk menyetelnya ke benar (true).
  1. Buka Safari > Preferensi.
  2. Pilih Keamanan.
  3. Pilih kotak centang Aktifkan JavaScript.
  1. Buka Setelan.
  2. Klik Lanjutan di sidebar kiri, lalu klik Privasi & Keamanan.
  3. Klik Setelan Situs
  4. Klik JavaScript.
  5. Di bagian atas, aktifkan Diizinkan.
