Aktifkan JavaScript untuk Melanjutkan

Untuk pengalaman terbaik, harap aktifkan JavaScript di pengaturan browser Anda.

Cara Mengaktifkan JavaScript:

Google Chrome:
  1. Klik pada ikon tiga titik di pojok kanan atas.
  2. Pilih "Settings".
  3. Scroll ke bawah dan klik "Privacy and security".
  4. Klik "Site settings".
  5. Di bagian "Content", cari "JavaScript" dan aktifkan.
Mozilla Firefox:
  1. Klik pada ikon tiga garis di pojok kanan atas.
  2. Pilih "Options".
  3. Klik "Privacy & Security".
  4. Scroll ke bawah ke bagian "Permissions" dan aktifkan JavaScript.
Microsoft Edge:
  1. Klik pada ikon tiga titik di pojok kanan atas.
  2. Pilih "Settings".
  3. Klik "Cookies and site permissions".
  4. Cari "JavaScript" dan aktifkan.
