Включите JavaScript и продолжите для доступа к сайту

JavaScript отключен в вашем браузере. Пожалуйста, включите JavaScript, чтобы получить доступ к этой странице.

  1. Нажмите на ⋮ и выберите Настройки.
  2. Перейдите в раздел Конфиденциальность и безопасность.
  3. Нажмите на Настройки сайтов.
  4. Нажмите на JavaScript.
  5. Выберите Разрешить выполнение JavaScript.

  2. Перейдите в раздел Файлы cookie и разрешения сайтов.
  3. Нажмите на JavaScript.
  4. Выберите Разрешить.

  2. Нажмите на Принять риски и продолжить.
  3. Найдите javascript.enabled.
  4. Нажмите на значение и установите его в true.
