启用 JavaScript 继续访问网站

您的浏览器禁用了 JavaScript。请启用 JavaScript 以继续访问网站。

  1. 点击 ⋮ 并选择 设置。
  2. 转到 隐私和安全。
  3. 点击 网站设置。
  4. 点击 JavaScript。
  5. 选择 允许网站使用 JavaScript。
  1. 点击 ⋮ 并选择 设置。
  2. 转到 Cookies 和站点权限。
  3. 选择 JavaScript。
  4. 启用 允许 (推荐)。
  1. 在地址栏中输入 about:config 并按回车。
  2. 点击“接受风险并继续”。
  3. 找到 javascript.enabled。
  4. 点击切换按钮，将其设置为 true。
  1. 打开 Safari > 偏好设置。
  2. 选择 安全性。
  3. 勾选 启用 JavaScript。
  1. 打开 设置。
  2. 在左侧边栏点击 高级，然后选择 隐私与安全。
  3. 点击 网站设置。
  4. 点击 JavaScript。
  5. 在顶部启用 允许。
